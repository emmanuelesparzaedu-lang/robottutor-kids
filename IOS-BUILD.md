# iOS preparación (debe ejecutarse en Mac)
# 1) En Mac: instalar Xcode + CocoaPods
# 2) Copiar este proyecto a Mac
# 3) Ejecutar:
#    npm install
#    npm install --prefix client
#    cd client
#    npm run build
#    npx cap add ios
#    npx cap sync ios
#    npx cap open ios
# 4) En Xcode: seleccionar Team, Bundle Identifier, Signing
# 5) Product > Archive > Distribute App > App Store Connect
