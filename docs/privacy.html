Política de privacidad — RoboTutor Kids

Última actualización: mayo de 2026

RoboTutor Kids es una aplicación educativa para niños. Esta política explica qué información se utiliza y cómo se protege.

1. Responsable

Desarrollador: Tec Stem / Emmanuel Esparza
Contacto: emmanuel.esparza.edu@gmail.com

2. Datos que se utilizan

  • Mensajes del chat: las preguntas que escribe o dicta el alumno para generar respuestas educativas.
  • Identificador anónimo de alumno: un código generado en el dispositivo para limitar el uso del servicio (no es el nombre real del niño).
  • Voz (opcional): si el usuario activa el micrófono, el audio se convierte a texto en el dispositivo; no guardamos grabaciones de voz en nuestro servidor.
  • Personalización local: nombre del asistente, colores del robot y dibujos se guardan en el dispositivo (almacenamiento local).

3. Datos que NO pedimos

No solicitamos nombre completo, dirección, teléfono, correo del niño, fotos ni ubicación GPS.

4. Cómo se procesan los datos

  • Los mensajes se envían de forma cifrada (HTTPS) a nuestro servidor en la nube.
  • El servidor aplica filtros de seguridad infantil antes y después de procesar la pregunta.
  • Las preguntas pueden procesarse con la API de Google Gemini para generar respuestas educativas, según las políticas de Google.

5. Uso por menores

La app está pensada para uso educativo con supervisión de un adulto (padre, madre o docente). Recomendamos que un adulto configure la app y supervise el uso.

6. Conservación y eliminación

  • El historial de chat en el dispositivo puede borrarse con el botón «Borrar» en la app.
  • Los datos en el servidor se usan solo para responder en la sesión y no se venden a terceros.

7. Terceros

Utilizamos servicios de:

  • Google Gemini API — procesamiento de lenguaje para respuestas educativas.
  • Render.com — alojamiento del servidor API (modo nube).

8. Seguridad

Aplicamos medidas técnicas como comunicación HTTPS, filtros de contenido inapropiado y límites de uso para proteger a los usuarios.

9. Cambios

Podemos actualizar esta política. La fecha de actualización aparecerá al inicio de esta página.
